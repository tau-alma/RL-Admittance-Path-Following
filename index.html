Admittance-Based Path Following for Excavators Using Model-Free Reinforcement Learning–Based Velocity Control

Amirmasoud Molaei1, Mohammad Heravi1, Reza Ghabcheloo1,
1Autonomous Mobile Machines Group, Faculty of Engineering and Natural Sciences, Tampere University, Finland
Paper arXiv Code

Basic block diagram of the algorithm.

Abstract

Autonomous motion control of hydraulic excavators in construction environments requires accurate path following in both free-space and contact-rich tasks such as grading. These operations involve nonlinear dynamics, model uncertainties, and complex interaction forces with terrain, which make conventional model-based velocity control difficult to tune and maintain across operating conditions. While classical control approaches rely heavily on accurate system identification, purely end-to-end learning methods often lack structure and interpretability. This paper presents an admittance-based path following framework with model-free reinforcement learning velocity control for a Komatsu PC138US excavator. The proposed architecture separates task-space path regulation from low-level velocity execution in a hierarchical manner. An outer-loop controller generates nominal end-effector velocity commands based on geometric path tracking, which are modulated through an admittance model to ensure compliant behavior during terrain interaction. An inner-loop model-free reinforcement learning policy, trained using Proximal Policy Optimization (PPO) in the AGX Dynamics® simulator, learns to track the commanded velocities under nonlinear hydraulic dynamics without explicit system modeling. The framework is evaluated on circular tracking and grading scenarios in both free-space and contact conditions. Extensive experiments demonstrate accurate path tracking, smooth velocity regulation, and stable interaction behavior under varying speeds and curvature conditions. The results highlight the effectiveness of combining structured admittance control with data-driven velocity learning for autonomous excavator operations.
